# Digital Control of Double Inverted Pendulum

## Overview
This project focuses on control-oriented modeling and stability analysis of an unstable system,
with emphasis on real-time implementation concepts used in embedded systems.

## Tools & Technologies
- MATLAB
- State-Space Modeling
- Control System Design

## Key Concepts
- System modeling
- Closed-loop control
- Stability analysis

## Engineering Focus
- Control logic suitable for embedded deployment
- Real-time constraints consideration
